## Getting Started

First, create a `.env.local` file based on [lib/env.ts](lib/env.ts)

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
